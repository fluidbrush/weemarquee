Gimme Code!

  1. Get the CSS

    This is the CSS you need for the plugin to work.

    If you don't copy the CSS to your main stylesheet, a few velocipractors will be dispatched to eat you.

  2. Get the JS

    Download the Javascript file and link to it (preferably before the closing body tag).

  3. Call it!

    $(".scroller").weemarquee();
